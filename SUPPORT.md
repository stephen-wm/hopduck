# Support

Thank you for your interest in HopDuck.

## Getting Help

If you need help using HopDuck or have questions about the project:

- Open a GitHub Discussion for general questions and ideas.
- Open a GitHub Issue if you believe you've found a bug.
- Review the documentation and README before opening a support request.

## Bug Reports

If you've found a bug, please open a GitHub Issue and include:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Environment details (OS, browser, Node.js version)

## Feature Requests

Feature requests and product ideas are welcome through GitHub Discussions or Issues.

## Security Issues

Please do not report security vulnerabilities through public GitHub Issues.

See [SECURITY.md](./.github/SECURITY.md) for instructions on responsible disclosure.

## Response Times

HopDuck is currently maintained by volunteers. Response times may vary depending on maintainer availability.
